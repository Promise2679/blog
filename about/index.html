关于

大家好，这里是Xenlyra，也可以叫我承诺，一个时常幻想能成为全能大神的普通人

梦想是有朝一日能随心所欲的完成自己想做的事情

名字的由来

其实一开始是没有 Xenlyra 这个名字的，我的网名就是承诺，但是注册域名的话，promise 的域名基本都注册光了，而且看上去也不美观，干脆就重新想一个英文 id 吧。于是经过几天的苦思冥想，我想到了这个没什么实际意义的英文名。

我是谁？

某双非高校大一在读，祖籍江西，但常年生活于山东。

技术栈：HTML，CSS，JavaScript，Vue，Golang，Gin，Gorm，MySQL，会一点 Python

成分：平台跳跃游戏玩家，即时战略游戏爱好者，音游彩笔，车万萌新，fps薯条弟

喜欢听音乐，杂食党什么都听

网站相关

网站使用 Hexo 博客框架搭建，主题是 Butterfly， 云部署在 Netlify 上

评论框架使用的是 Twikoo，原因是觉得他比较简洁美观

网站中的大部分图片都存放在 Cloudflare R2 中

如果要发表评论，使用 qq 邮箱会自动获取头像，其他邮箱或者想自定义头像请到 Cravatar 上配置

如果想联系我，请使用邮件：i@xenlyra.cc
